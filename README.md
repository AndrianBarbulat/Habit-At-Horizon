# 🌅 Habit At Horizon

**React Native Mobile App for Mentorship, Habit Formation, and Collaborative Skill Development**

---

## 📖 About

Habit at Horizon is a mobile application that connects mentors and mentees in a warm, personal learning environment. The app creates a space where people can share knowledge, build habits, and develop skills together — like a friendly neighborhood learning club where everyone helps each other grow.

**Mentors** share their expertise by creating personalized lessons and activities, guiding learners through real-world skills — from coding and team management to hands-on hobbies.

**Mentees** find the right person to help them with whatever they're curious about, in a relaxed and supportive atmosphere that feels like asking a friend for help rather than searching the internet.

## 🚀 Getting Started

### Prerequisites

- Node.js installed
- JDK 17 (e.g. [Eclipse Adoptium](https://adoptium.net/))
- Android Studio with an emulator or a physical Android device
- Firebase project with `google-services.json`

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/GuineaCoding/Habit-At-Horizon.git
   cd Habit-At-Horizon/HabitAtHorizonMobileApp
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Add your `google-services.json` file to `android/app/`

4. Configure Java path in `android/gradle.properties`:
   ```
   org.gradle.java.home=C:/Program Files/Eclipse Adoptium/jdk-17.x.x-hotspot
   ```

5. Run the app:
   ```bash
   npx react-native run-android
   ```
